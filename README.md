# Adivinhando Número
Nesse jogo voce deve adivinhar o numero aleatorio gerado, se adivinhar voce ganha o jogo.

## Utilizando o básico de DOM js
Foi feito autilozacao de alguns elementos do dom, exemplo: 
- ClassList
- toggle
- document.querySelector
- value
- innerText
- remove
- add

## eventos
- addEventListener()
- preventDefault
